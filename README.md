# KaseiCoin


![joint_savings_hero](images/KASEI.jpg)

This smart contract principal function is to automate financial processes, such as hosting joint savings accounts.

Solidity smart contract that accepts two user addresses. These addresses will be able to control a joint savings account. The smart contract will use ether management functions to implement a financial institutionâ€™s requirements for providing the features of the joint savings account. These features will consist of the ability to deposit and withdraw funds from the account.
